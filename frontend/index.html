How It Works
01
Any Wallet
Paste any wallet address. Skoré instantly analyses on-chain history across 3 chains — no login, no MetaMask required.
02
Oracle Scores
Kwala detects the ScoreRequested event and triggers our API to analyse wallet history across Ethereum, Polygon & Base.
03
SBT Minted
Your score (300–850) is written back on-chain and minted as a non-transferable Soulbound Token directly to your wallet.
04
Lenders Query
DeFi protocols and fintechs query your SBT score via API to make lending decisions — without a bank account or credit bureau.
